# Правильные Шахматы

## Учебный проект 4го месяца обучения

## Описание 
Приходите к нам, играть в шахматы! Сразитесь классическом поединке, блиц состязании и бою с гройсмейстером. 

Практика в SCSS

## Инструменты
* HTML разметка страницы
* SCSS стили
* Структуировано по БЭМ
* Mobile first подход
* JavaScript
